# Words Counter App

A modern and fast Words Counter application built with React and Vite. This tool helps users count characters, words, and check text limits for popular social media platforms like Instagram and Facebook.

---

## Features

- Character Counter (with and without spaces)
- Word Counter
- Instagram Caption Limit Checker
- Facebook Post Limit Checker
- Real-time text analysis
- Clean and simple user interface
- Fast performance using Vite
- Responsive design

---

## Use Case

This app is useful for:

- Content creators
- Social media managers
- Students
- Copywriters
- Freelancers
- Anyone writing captions, posts, or paragraphs

Users can check whether their text exceeds Instagram or Facebook limits before posting.

---

## Tech Stack

- React 19
- Vite 7
- JavaScript (ES Modules)
- CSS
- ESLint
